## ⚙️ Tecnologias Utilizadas

| Tecnologia | Propósito |
|------|------|
| [Astro 6](https://astro.build) | Geração de site estático (SSG) + View Transitions |
| [Tailwind CSS v4](https://tailwindcss.com) | Framework CSS (`@theme` syntax) |
| [Shiki](https://shiki.style) | Highlighting de sintaxe de código (ideal para scripts e exploits) |
| [MDX](https://mdxjs.com) | Suporte a Markdown com componentes JSX injetados |
| [Waline](https://waline.js.org) | Sistema de comentários |
